# Day 6 – Triggers & SOQL

## What is SOQL?

SOQL is Salesforce Object Query Language used to retrieve data from Salesforce objects.

---

## What is an Apex Trigger?

An Apex Trigger is code that runs automatically when records are inserted, updated, deleted, or undeleted.

---

## Difference Between Flow vs Trigger

| Flow | Trigger |
|------|---------|
| No-code | Code-based |
| Easy to build | Complex logic |
| Best for simple tasks | Best for advanced automation |

---

## Difference Between Before vs After Trigger

| Before Trigger | After Trigger |
|----------------|---------------|
| Runs before save | Runs after save |
| Used for validation | Used for related records |

---
