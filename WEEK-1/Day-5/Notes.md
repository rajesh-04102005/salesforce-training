# Notes - Day 5

## What is Apex?
Apex is a cloud-based programming language used in Salesforce.

## Why Apex is Needed?
- Complex business logic
- External integrations
- Advanced automation

## Declarative Development
Uses clicks, flows, and configuration.

## Programmatic Development
Uses Apex code for advanced logic.

## DML Operations
- Insert
- Update
- Delete
- Upsert

## SOQL
Used to query Salesforce records.

Example:
SELECT Name FROM Account
