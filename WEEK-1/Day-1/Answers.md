# 📝 Mandatory Submission Answers

## 1. What problem does Salesforce solve?

Salesforce solves the problem of managing customer relationships and business processes efficiently. It helps companies organize customer data, automate workflows, improve sales, and provide better customer support.

---

## 2. What is CRM?

CRM stands for **Customer Relationship Management**. It is a system used to store customer information, track interactions, manage sales, and improve customer relationships.

---

## 3. What is an Object in Salesforce? Give one example.

An object in Salesforce is like a table in a database that stores a specific type of information.

### Example:
- **Contact Object** – stores customer contact details.

---

## 4. Difference between Salesforce Admin and Developer

| Salesforce Admin | Salesforce Developer |
|------------------|----------------------|
| Works mostly without coding | Uses programming |
| Configures Salesforce | Builds custom applications |
| Uses declarative tools | Uses Apex and LWC |
| Manages users and security | Develops advanced features |

---

## 5. Suggest one real-world application that can be built using Salesforce.

A **College Management System** can be built using Salesforce to manage:

- Students
- Faculty
- Courses
- Attendance
- Fees
- Results
