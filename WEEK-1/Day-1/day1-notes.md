1️⃣ Introduction to Salesforce

Today I learned the fundamentals of Salesforce Platform and understood how businesses use Salesforce to manage customers, automate processes, and build applications.

2️⃣ What is Salesforce?

Salesforce is a cloud-based CRM platform used by companies to:

- Manage customer relationships
- Track sales
- Store customer data
- Automate business processes
- Build custom applications

It is not just a CRM tool but also a powerful development platform.

Key Points:
- Works on the cloud
- No software installation required
- Accessible from anywhere
- Supports automation and app development

3️⃣ What is CRM?

CRM stands for Customer Relationship Management.

CRM helps companies:
- Store customer information
- Track customer interactions
- Improve communication
- Manage leads and sales
- Provide customer support

Example:
When a customer contacts a company, the following information can be stored in Salesforce:
- Customer details
- Purchase history
- Support tickets
- Emails

4️⃣ Why Companies Use Salesforce

Companies use Salesforce because it helps them:
- Increase sales
- Improve customer service
- Automate repetitive tasks
- Organize business data
- Improve productivity
- Make data-driven decisions

Salesforce provides:
- Sales tools
- Marketing tools
- Customer support tools
- AI tools
- Reporting and dashboards

5️⃣ What is a Salesforce Developer?

A Salesforce Developer:
- Builds custom applications
- Creates automation
- Writes Apex code

- Customizes Salesforce for business needs

Skills Required:
- Problem solving

7️⃣ Salesforce Data Model Basics

A. Objects
Objects are like tables in a database.
They store a specific type of information.

Examples:
- Contact
- Account
- Lead
- Opportunity

B. Records
Records are rows inside objects.

Example:
A student inside the Student object is one record.

C. Fields
Fields are columns inside objects.

Examples:
- Name
- Email
- Phone Number

8️⃣ Standard vs Custom Objects

Standard Objects:
Already provided by Salesforce.

Examples:
- Account
- Contact
- Lead

Custom Objects:
Created by users for business-specific needs.

Examples:
- Property
- Student
- Course

🔟 What is Metadata?

Metadata means “Data about data”.

Examples:
- Field names
- Object structure
- Page layouts
- Security settings

Metadata defines the structure of Salesforce.

1️⃣1️⃣ What is Data 360?

Data 360 means a complete connected view of business/customer data.

It combines data from:
- Salesforce
- External systems
- Websites
- Apps

Benefits:
- Better AI
- Better customer understanding
- Real-time insights

1️⃣2️⃣ Declarative vs Programmatic Development

Declarative (No-Code):
Build using:
- Drag-and-drop
- Setup menus
- Forms

Examples:
- Create fields
- Flows
- Validation rules

Programmatic Development:
Build using code.

Uses:
- Apex
- Lightning Web Components
