1️⃣ What is Salesforce?

Salesforce is a cloud-based CRM platform used by companies to:

Manage customer relationships
Track sales
Store customer data
Automate business processes
Build custom applications

It is not just a CRM tool but also a powerful development platform.

Key Points
Works on the cloud
No software installation required
Accessible from anywhere
Supports automation and app development

6️⃣ Salesforce Data Model Basics
A. Objects

Objects are like tables in a database.

They store a specific type of information.

Examples
Contact
Account
Lead
Opportunity
B. Records

Records are rows inside objects.

Example:
A student inside the Student object is one record.

C. Fields

Fields are columns inside objects.

Example:

Name
Email
Phone Number
7️⃣ Standard vs Custom Objects
Standard Objects

Already provided by Salesforce.
Examples:

Account
Contact
Lead
Custom Objects

Created by users for business-specific needs.
Examples:

Property
Student

9️⃣ What is Metadata?

Metadata means:

“Data about data”

Example:

Field names
Object structure
Page layouts
Security settings

Metadata defines the structure of Salesforce.

Data 360 means:

A complete connected view of business/customer data.

It combines data from:

External systems
Websites
Apps

Benefits:

Better AI

Real-time insights
