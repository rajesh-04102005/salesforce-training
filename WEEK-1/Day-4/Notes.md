# Notes – Salesforce Flow Builder

## Key Points
- Salesforce Flow automates business processes.
- Flow Builder is a declarative tool.
- Variables store temporary data.
- Data Elements work with records.
- Action Elements send emails and notifications.

## Important Elements

| Element | Purpose |
|---|---|
| Get Records | Retrieve records |
| Create Records | Create records |
| Update Records | Update records |
| Delete Records | Delete records |
| Action | Send notifications |

## Best Practices
- Add descriptions to flows
- Use clear labels
- Test flows before activation
- Avoid unnecessary deletions

## Benefits of Automation
- Saves time
- Improves productivity
- Reduces manual work
- Improves accuracy
