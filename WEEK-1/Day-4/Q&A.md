# Questions and Answers

## 1. Why do companies automate workflows?
To save time and improve productivity.

## 2. Problems with manual processes?
- Human errors
- Delays
- Repetitive work

## 3. Difference between Screen Flow and Record-Triggered Flow?

| Screen Flow | Record-Triggered Flow |
|---|---|
| Requires user interaction | Automatic |
| Uses forms | Runs in background |

## 4. Why is no-code automation powerful?
It allows automation without programming.

## 5. When should automation be avoided?
When processes require constant human judgment.

## 6. How does automation improve consistency?
Automation performs tasks the same way every time.
