# Salesforce Administrator Track - Day 2
## Reports and Dashboards

## What is a Report?

A Salesforce Report is a list of records that meet specific criteria. Reports help users organize, analyze, and summarize data stored in Salesforce.

### Features of Reports
- Display records from Salesforce objects
- Apply filters to narrow down data
- Group and summarize information
- Generate charts and graphs
- Export data for analysis

### Example
A Student Attendance Report showing students with attendance below 75%.

---

## What is a Dashboard?

A Dashboard is a visual representation of report data. It displays charts, metrics, graphs, and key performance indicators (KPIs) that help management quickly understand business performance.

### Features of Dashboards
- Visual presentation of data
- Real-time business insights
- Multiple report components in one place
- Easy performance tracking
- Supports management decision-making

### Example
A Principal Dashboard displaying attendance, placements, internships, and student performance.

---

## Difference Between Reports and Dashboards

| Reports | Dashboards |
|----------|-----------|
| Display detailed data | Display summarized visual data |
| Can be exported | Cannot directly show detailed records |
| Used for analysis | Used for monitoring |
| Tabular or summarized format | Charts, graphs, and metrics |
| Source of data for dashboards | Built using reports |

# Reflection

Reports and dashboards help organizations transform raw data into meaningful information. Instead of manually checking records, management can quickly monitor important metrics and identify trends. This improves efficiency and enables better planning.

Through Salesforce reports and dashboards, administrators can provide decision-makers with accurate, real-time insights that improve organizational performance.

---

# How Does Data Help Management Make Decisions?

Data helps management by:

1. Identifying performance trends.
2. Monitoring attendance levels.
3. Tracking placement performance.
4. Measuring internship participation.
5. Evaluating department performance.
6. Detecting problem areas early.
7. Allocating resources efficiently.
8. Setting realistic goals.
9. Improving student outcomes.
10. Supporting data-driven decision making.

### Example

If attendance drops significantly, management can investigate causes and take corrective actions. If placement percentages decrease, additional training programs can be introduced.

Therefore, reports and dashboards convert data into actionable insights that support informed decision-making.
