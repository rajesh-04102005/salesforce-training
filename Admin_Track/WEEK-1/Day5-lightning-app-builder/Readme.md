# Lightning App Builder

## What is Lightning Experience?

Lightning Experience is the modern Salesforce user interface that provides a more productive, intelligent, and user-friendly environment compared to Salesforce Classic. It offers features such as customizable pages, dashboards, reports, Kanban views, and Lightning Components that improve user productivity and business efficiency.

### Benefits of Lightning Experience

* Modern and responsive interface
* Improved user productivity
* Enhanced reporting and dashboards
* Mobile-friendly design
* Easy customization without coding
* Better collaboration and data visualization

---

## What is Lightning App Builder?

Lightning App Builder is a drag-and-drop tool that allows Salesforce Administrators to create and customize pages without writing code.

Using Lightning App Builder, administrators can build:

* Home Pages
* Record Pages
* App Pages

It helps organizations create personalized user experiences by arranging components, fields, reports, dashboards, and actions according to business requirements.

### Features

* Drag-and-drop customization
* Dynamic page visibility
* Reusable Lightning components
* Mobile and desktop support
* Profile-based page assignments
* No-code page development

# Reflection

## Why should different users have different user interfaces?

Different users perform different tasks within an organization. Providing the same interface to everyone can create confusion, reduce productivity, and expose unnecessary information.

Customized user interfaces allow users to focus only on the information and actions relevant to their responsibilities.

### Advantages

* Improved productivity
* Simplified navigation
* Better user experience
* Reduced training effort
* Faster decision-making
* Increased data security
* Personalized workflows

Therefore, different user interfaces help organizations deliver the right information to the right users at the right time.
